# AssetLock

Time-locked asset vesting contract.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/AssetLock.sol`
- Stacks: `contracts/stacks/contracts/assetlock.clar`

## Setup
```bash
pnpm install
pnpm dev
```
